### 🎭 動畫效果

**淡入動畫 (fadeIn)**
- 從下方 10px 淡入
- 持續時間: 0.5s

**懸浮效果**
- 放大: scale(1.02)
- 過渡: 300ms
- 按下: scale(0.98)

**載入動畫**
- 旋轉載入: animate-spin
- 脈衝效果: animate-pulse

### 🚀 如何使用

1. **開發模式**
   ```bash
   npm run dev
   # 訪問 http://localhost:5173/sidepanel.html
   ```

2. **建置擴充功能**
   ```bash
   npm run build
   # 產生 dist/ 資料夾
   ```

3. **載入到 Chrome**
   - 打開 `chrome://extensions/`
   - 啟用「開發者模式」
   - 點擊「載入未封裝項目」
   - 選擇 `dist/` 資料夾
